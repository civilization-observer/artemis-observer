# Third-party assets and runtime files

This project vendors a small set of browser runtime files and visual assets so the GitHub Pages app can load its primary scene without relying on external CDNs at runtime.

## Runtime

- Three.js 0.160.0 (`vendor/three/*`) - MIT License
- satellite.js 6.0.2 (`vendor/satellite/satellite.min.js`) - MIT License

## Textures

- Earth day, bump, and night textures from the public `three-globe` examples.
- Earth cloud texture from `turban/webgl-earth`.
- Moon texture from Wikimedia Commons: `Solarsystemscope texture 2k moon.jpg`, by Solar System Scope, CC BY 4.0.
